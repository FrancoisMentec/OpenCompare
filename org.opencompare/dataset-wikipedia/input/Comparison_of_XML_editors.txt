==NOTE==
Spring Tools Suite and Eclipse have a variety of built-in and free plugins matching or exceeding the quality of the paid versions. They come with a content assist tool that completes tags and can search for classes in any java classpath. They also include validation, bean creation, and commit tools.

A plugin for [[Notepad++]] named XML Tools is available. It contains many features including manual/automatic validation of both DTDs and XSDs, XPath evaluation, auto-completion, pretty print, and text conversion in addition to being able to work on multiple files at once. Other tools are available to run xhtml.
